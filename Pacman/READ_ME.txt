Voici la notice pour pouvoir modifier et générer son propre labyrinthe :

(Vous pouvez directement modifiez le fichier labyrinthe.txt, mais aucune 
sauvegarde du labyrinthe par défaut n'est présente dans le reste du logiciel,
pensez à le sauvergarder sur un autre fichier ou a renommer celui ci)

Dans un fichier labyrinthe.txt :

Sur la première ligne écrivez le nombre de colonnes que va avoir votre 
labyrinthe.
Ensuite modélisez-le en suivant ces régles de syntaxe :
Chaque caractère représente une case.

S => le spawn de Pac-Man
s => un espace dans lequel se trouve une gum
G => un espace dans lequel se trouve une Super-gum
w => un mur
e => un espace vide
c => un spawn pour un UNIQUE fantôme
